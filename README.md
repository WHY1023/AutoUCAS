# AutoUCAS 课程自动签到助手
*(I Work for Progress / I Want to Play)*

这是一个基于 Python 和 Tkinter 开发的轻量级桌面 GUI 应用程序，专为中国科学院大学（UCAS）学生设计。它可以帮助你在后台静默监听课程时间表，并在设定的时间自动完成课程签到。

> **免责声明**：本项目及代码仅供 Python 编程、GUI 开发及网络请求学习交流使用，请勿用于违反学校规章制度的用途。

## ✨ 核心特性

* **现代图形界面**：基于 Tkinter 深度定制的扁平化界面，包含动态呼吸灯彩蛋标题。
* **多线程架构**：网络请求与 UI 渲染分离，挂机守护进程在后台独立运行，主界面永不卡顿。
* **智能断线重连**：自动处理会话（Session）过期问题，401 状态下自动重新登录并恢复任务。
* **富文本日志**：提供直观的彩色运行日志（成功/失败/警告），带有精确到秒的时间戳。
* **灵活配置**：
  * 支持自定义**提前签到时间**（建议 20 分钟）。
  * 支持自定义**课表刷新间隔**（建议 1 分钟）。

## 🛠️ 环境依赖

* **Python**: 3.7+ (需包含标准的 `tkinter` 库)
* **第三方库**: `requests`
